## 权限模型（修正核心）

```
┌──────────────────────────────────────────┐
│              角色 (UserRole)               │
│  Teacher  │  Student  │  Admin            │
│  控制：能做什么操作（功能权限）              │
└──────────────┬───────────────────────────┘
               │
               ▼
┌──────────────────────────────────────────┐
│         数据可见性（业务归属）               │
│  Teacher → 自己的课程/作业/学生             │
│  Student → 自己的提交/所在课程作业           │
│  Admin  → 全部数据                         │
└──────────────┬───────────────────────────┘
               │
               ▼
┌──────────────────────────────────────────┐
│     管理体系 (ManagementSystem)            │
│  自定义字段模板 + 记录分组                  │
│  每个用户有一个默认体系，教师可创建自定义体系  │
│  非全局隔离容器，不影响核心业务数据权限       │
└──────────────────────────────────────────┘
```
